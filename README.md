# weather_app

A simple weather app that fetches current weather conditions and forecasts using the OpenWeatherMap API.

Features

- Displays current temperature, humidity, and weather conditions
- Supports multiple locations
- Responsive design for mobile and desktop devices

API Used

- OpenWeatherMap API

Usage

1. Enter a location in the search bar
2. Click the "searchbar" button to fetch the current weather conditions and forecast
